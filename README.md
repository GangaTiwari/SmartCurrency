SmartCurrency

SmartCurrency is a web application that displays real-time prices of popular cryptocurrencies (Bitcoin, Ethereum, and Dogecoin) using the CoinGecko API. The project features a modern, responsive design with a clean user interface, built using HTML, CSS, and JavaScript with jQuery.

Features





Real-time Crypto Prices: Fetches and displays current prices for Bitcoin, Ethereum, and Dogecoin in USD.



Responsive Design: Optimized for various screen sizes with a visually appealing layout.



Interactive Navigation: Includes a navigation bar with hover effects and a call-to-action button.



Crypto Information Section: Displays cryptocurrency details with icons and prices in a sleek format.

Technologies Used





HTML5: Structure of the web page.



CSS3: Styling with a custom design, including background images and hover effects.



JavaScript & jQuery: Handles API requests to fetch and display cryptocurrency prices.



CoinGecko API: Provides real-time cryptocurrency price data.

Setup Instructions





Clone the Repository:

git clone https://github.com/your-username/smartCurrency.git



Navigate to the Project Directory:

cd smartCurrency



Open the Project:





Open index.html in a web browser to view the application.



Ensure an internet connection is available for the CoinGecko API to fetch data.

File Structure





index.html: Main HTML file containing the structure of the web page.



Styles.css: CSS file for styling the layout, navigation, and crypto price display.



script.js: JavaScript file that fetches and updates cryptocurrency prices using the CoinGecko API.



images/: Directory containing the background image (background.png) and cryptocurrency icons.

Usage





The navigation bar includes links for "Market," "Features," "White Papers," and "About Us."



The main section displays a heading, description, and a "Get Started" button.



The bottom-right section shows real-time prices for Bitcoin, Ethereum, and Dogecoin, fetched from the CoinGecko API.

Dependencies





jQuery: Included via CDN in index.html for handling API requests.



CoinGecko API: No API key required for the free tier used in this project.

Notes





Ensure the images folder contains the required background.png and cryptocurrency icons for proper rendering.



The CoinGecko API is used to fetch live data; ensure a stable internet connection.



The project assumes the browser supports modern JavaScript and CSS features.
